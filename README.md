# atividade-2-
O objetivo desta tarefa é realizar uma simulação no Wokwi utilizando um ESP32 para  ler o estado de três botões e acionar LEDs seguindo a lógica abaixo: ● ● ● Botão 1 pressionado: Acende apenas o LED Verde. Botão 2 pressionado: Acendem os LEDs Verde e Amarelo. Botão 3 pressionado: Acendem todos os LEDs (Verde, Amarelo e Vermelho).
